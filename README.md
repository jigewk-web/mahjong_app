# 麻雀点数記録アプリ

麻雀の対局結果を記録・集計できるWebアプリです。  
プレイヤーごとの点数入力、順位計算、対局履歴の保存ができます。

Laravelを用いてCRUD処理やリレーション設計を実装し、Dockerで開発環境を構築しました。

---

## 使用技術

- PHP 8.1
- Laravel 10.x
- MySQL 8.0
- Docker / Docker Compose
- Blade
- HTML / CSS
- Git / GitHub

---

## 主な機能

- プレイヤー登録
- 対局作成
- 点数入力
- 対局結果一覧表示
- 対局結果編集
- 対局削除

---

## テーブル設計

- players
- games
- game_players

複数プレイヤーと対局結果を管理するため、中間テーブル `game_players` を使用してリレーションを構築しています。

---

## 工夫した点

### 1. リレーション設計
1回の対局に対して複数のプレイヤーが参加できるように、`games` と `players` の間に `game_players` テーブルを作成し、多対多のリレーションを実装しました。

### 2. 点数計算の自動化
各プレイヤーの順位・点数を入力すると順位を元に設定済みのルールより自動でポイントが計算されるように実装しました。

### 3. CRUD機能の実装
対局結果の登録・編集・削除を実装し、基本的なデータ管理機能を備えています。

### 4. Dockerによる環境構築
Docker Composeを用いて、PHP・MySQL環境を構築し、環境差異が出ないようにしました。
